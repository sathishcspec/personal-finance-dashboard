# 💰 Personal Finance Analytics Dashboard | Power BI

An interactive **Power BI dashboard** that provides a comprehensive overview of personal financial performance. This dashboard enables users to monitor income, expenses, savings, investments, cash flow, and credit score through intuitive visualizations and KPIs, helping support better financial decision-making.

---

## 📌 Project Overview

The **Personal Finance Analytics Dashboard** transforms raw financial data into meaningful insights using Power BI. It combines interactive reports, KPI cards, and dynamic filters to help users analyze spending habits, track savings goals, and evaluate overall financial health.

---

* 📈 Track Monthly Income vs. Expenses

## 🚀 Getting Started

1. Clone this repository.

```bash
git clone https://github.com/your-username/Personal-Finance-Dashboard.git
```

2. Open **personal.pbit** in **Microsoft Power BI Desktop**.

3. Connect the template to your dataset if prompted.

4. Refresh the data.

5. Explore the dashboard using the available filters and visuals.
